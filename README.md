# Instalaltion

If you device is Windows , you must download [python2](https://python.org)
Then you type : 
- `cd WebDav-manual`
- `python2 start.py`

If you device is Linux , you must clone this repo :)
Then you type :
- `cd WebDav-manual`
- `bash run.sh`
